# Pokemon App

The Pokemon App is a Frontend app made with React, Redux appliying [Duck methodology](https://github.com/erikras/ducks-modular-redux) and using [Pok√©Api](https://pokeapi.co/).

## Getting Started

The first step is install all project dependencies

### Installing all dependencies

Runs with npm

```
npm install
```

Run with yarn

```
yarn
```

## Running the project

Runs the development mode with following comands

Run with npm

```
npm run start
```

Run with yarn

```
yarn start
```
